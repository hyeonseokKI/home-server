## TypeScript란?

TypeScript는 JavaScript에 타입 시스템을 추가한 언어입니다.

## 타입 안전성

타입 안전성은 컴파일 타임에 오류를 발견할 수 있게 해줍니다.

### 기본 타입

- `string`: 문자열
- `number`: 숫자
- `boolean`: 불리언
- `object`: 객체

## 인터페이스 사용하기

인터페이스를 사용하여 객체의 구조를 정의할 수 있습니다.

```typescript
interface User {
  name: string;
  age: number;
  email: string;
}
```

## 제네릭

제네릭을 사용하면 재사용 가능한 타입을 만들 수 있습니다.

## 결론

TypeScript를 사용하면 더 안전하고 유지보수하기 쉬운 코드를 작성할 수 있습니다.
