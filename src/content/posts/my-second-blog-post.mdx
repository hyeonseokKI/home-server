## Tailwind CSS 소개

Tailwind CSS는 유틸리티 우선 CSS 프레임워크입니다.

## 반응형 디자인

반응형 디자인은 다양한 화면 크기에 대응하는 디자인 기법입니다.

### 브레이크포인트

- `sm`: 640px
- `md`: 768px
- `lg`: 1024px
- `xl`: 1280px

## 유틸리티 클래스

Tailwind CSS는 유틸리티 클래스를 제공합니다.

### 예시

```tsx
<div className="flex flex-col md:flex-row gap-4">
  <div className="w-full md:w-1/2">콘텐츠</div>
</div>
```

## 결론

Tailwind CSS를 사용하면 빠르고 일관된 디자인을 구현할 수 있습니다.
